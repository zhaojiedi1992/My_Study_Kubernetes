3.2. 容器环境

3.2.1. 容器环境内容

容器环境内容有如下几个部分

  • 文件系统，一个镜像和多个卷

  • 容器自身信息

  • 集群其他对象信息

3.2.2. 容器信息

容器信息，pod名称和命名空间可以通过下行api转换为环境变量 样例todo

3.2.3. 集群信息

创建容器时正在运行的所有服务都可以作为该容器的环境变量，对于名为A的服务，当映射到名为B的容器时，

A_SERVICE_HOST=<the host the service is running on>
A_SERVICE_PORT=<the port the service is running on>
